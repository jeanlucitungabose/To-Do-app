# To-Do-app
Here it is to do app which designed to help peaple acroos the globe to track and manage their task easier
Research Findings
Based on research about productivity applications:
Students prefer simple interfaces over complex systems
Visual task tracking improves motivation.
Lightweight web applications are more accessible.
Saving data locally allows offline usability.
This project implements these findings by keeping the interface minimal and easy to use.
